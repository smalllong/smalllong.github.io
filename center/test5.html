Centering using a full height sibling: #parent { text-align: center; } #parent::before { content: ''; height: 100%; } #parent::before, #child { display: inline-block; vertical-align: middle; } Pros: can see full content when child is larger than parent Cons: complex CSS to prevent ghost white space, child need some hacks (put child close to parent in html, or set font-size to 0)
